# sphereql-index

Spatial indexing for the [sphereQL](https://github.com/bkahan/sphereQL) project.

Partitions S² into radial shells and angular sectors for fast spatial queries: cone, cap, shell, band, wedge lookups and k-nearest-neighbor search. Includes cached Cartesian vectors for fast angular distance proxy.

See the [main repository](https://github.com/bkahan/sphereQL) for full documentation, examples, and architecture overview.
