# sphereql-graphql

GraphQL integration for the [sphereQL](https://github.com/bkahan/sphereQL) project.

Provides an `async-graphql` schema with queries for cone, shell, band, wedge, and region lookups, k-nearest-neighbor search, distance calculations, and real-time subscriptions via a broadcast event bus.

See the [main repository](https://github.com/bkahan/sphereQL) for full documentation, examples, and architecture overview.
