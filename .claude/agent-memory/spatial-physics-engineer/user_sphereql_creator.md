Ben is the creator and primary developer of sphereQL, a multi-crate Rust workspace for projecting high-dimensional embeddings onto a 3D sphere. The project spans core geometry, spatial indexing, layout, embedding pipelines, GraphQL, WASM/Python bindings, and vector DB integrations. Ben works at a senior level across Rust, has strong interest in the mathematical foundations (PCA, kernel PCA, spherical geometry), and is exploring how the Category Enrichment Layer could serve as an AI reasoning substrate. He values production-quality code with physics-informed rigor and dislikes speculative abstractions.
